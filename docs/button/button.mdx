## Icon Button

In order to render a button which includes an icon, simply pass the icon element you with to render with the `icon` prop. You can also customize the position of the icon with the `iconPos` prop.

<Playground>
  <Button 
    icon={<Download color="white" size={20} />} 
    text='Download' 
    bg={Colors.success} 
  />
</Playground>

## Custom Width

By default the `Button` component is set to a minimum width of `120px` with an automatic width. If you would like to pass a custom width, you can use the `width` prop.

<Playground>
  <Row>
    <Button width='30%' text='Click Me' bg={Colors.blue} />
    <Button width='30%' text='Success' bg={Colors.success} />
    <Button width='30%' text='Error'  bg={Colors.error} />
  </Row> 
</Playground>

## Different Sizes

There are 3 pre-defined sizes for the `Button` component, pass either `sm`, `md`, or `lg` to the size prop to resize the buttons.

<Playground>
  <Row>
    <Button size='sm' width='30%' text='Click Me' bg={Colors.blue} />
    <Button size='md' width='30%' text='Click Me' bg={Colors.blue} />
    <Button size='lg' width='30%' text='Click Me' bg={Colors.blue} />
  </Row> 
</Playground>
