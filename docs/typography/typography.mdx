# Typography

** Documentation and examples for Everblue UI typography, including global settings, headings, body text, and more. **

All the text components provided by Everblue adopt responsive sizing using the `FluidType` component, which calculates the correct size of the text based on the viewport width and doesn't rely on staggered media queries.
